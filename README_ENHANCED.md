## 新增功能详解

### 1. 环节级成本查询 (CostQueryEngine)

支持查询9大环节的详细成本构成：

```
用户: 运输成本为什么这么高？

系统:
=== 运输配送成本详情 ===

【成本汇总】
总成本: ¥10,650.00
占比: 15.23%

【成本构成】
  • 变动成本（燃油）: ¥2,625.00 (24.6%)
    计算公式: 750公里 × 3.5元/公里
    说明: 车辆燃油费用

  • 固定成本（折旧）: ¥3,600.00 (33.8%)
    计算公式: 120元/天 × 30天
    说明: 车辆折旧、保险等

【成本洞察】
  💡 该环节成本占比15.2%，属于中等成本项
  💡 运输成本与配送距离和频次直接相关，可考虑路线优化
```

**支持的查询类型：**
- 环节成本详情（订单处理、运输配送、末端交付等）
- 组件成本查询（上楼费、冷链包装、燃油费等）
- 成本对比分析
- 假设分析（What-If Analysis）

### 2. 数据驱动的建议 (SuggestionEngine)

每条建议都包含具体数据支持：

```
🔴 建议1: 优化配送路线，提升装载率
   类别: 运输优化 | 实施难度: 中等 | 优先级: 高

   问题描述: 当前运输成本占比过高，通过路线优化和装载率提升可显著降低成本

   📊 数据支持: 运输成本占比35.2%，高于行业平均25%。通过路线优化可减少约12%的行驶里程

   💰 预期节省: ¥12,780.00/月 (12.0%)

   📝 行动步骤:
      • 使用TMS系统进行路线规划
      • 合并相近区域订单，提高单车装载率至85%以上
      • 设置配送时间窗口，减少等待时间
```

**建议来源：**
- 成本结构分析（占比>30%的环节）
- 业务特征分析（单均件数、配送距离等）
- 行业基准对比（TOB: ¥50-80/单，餐配: ¥80-120/单）

### 3. 智能追问处理 (FollowUpHandler)

支持多种类型的追问：

#### 3.1 环节成本查询
```
用户: 末端交付成本为什么这么高？
系统: [输出末端交付详细成本构成和计算公式]
```

#### 3.2 假设分析
```
用户: 如果日订单增加到150单会怎样？

系统:
【假设分析结果】

参数调整: {'monthly_order_count': 4500}

成本变化:
  原成本: ¥252,736.68/月
  新成本: ¥298,450.32/月
  差异: +¥45,713.64/月 (+18.1%)

新单均成本: ¥66.32
可行性评级: medium

结论: 调整后成本增加18.1%
```

#### 3.3 组件查询
```
用户: 上楼费具体是多少？

系统:
【上楼费详情】
所属类别: 末端交付
总成本: ¥9,000.00
占比: 3.56%

成本构成:
  • 上楼费用: ¥9,000.00
    计算公式: 900层 × 10元/层
```

#### 3.4 一般性咨询
```
用户: 这个成本在行业里算什么水平？

系统: [基于知识库的行业基准数据回答]
```

### 4. 知识库系统 (KnowledgeBase)

内置行业知识和最佳实践：

**知识分类：**
- `industry_benchmark` - 行业基准数据
- `best_practice` - 最佳实践案例
- `risk_case` - 风险案例
- `optimization_tip` - 优化技巧

**默认知识条目：**
1. TOB企业购单均成本基准（¥50-80）
2. 餐配业务单均成本基准（¥80-120）
3. 配送路线优化最佳实践
4. 库存周转率提升方法
5. 上楼配送成本风险案例
6. 冷链断链风险案例
7. 批量处理降本技巧
8. 订单合并策略

**知识管理功能：**
- 添加/更新/删除知识
- 按类别/标签/关键词搜索
- 基于成本结构自动推荐相关知识
- 使用统计和热度排序

### 5. 增强版评估流程

```
用户输入
    ↓
意图识别 → 实体抽取 → 参数校验
    ↓
成本计算（9大环节）
    ↓
数据驱动建议生成
    ↓
相关知识推荐
    ↓
增强版报告输出
    ↓
追问模式（支持多轮交互）
```

## 技术亮点

1. **成本透明化**
   - 每个成本环节都有详细的计算公式和说明
   - 支持组件级成本查询
   - 提供成本洞察和优化方向

2. **数据驱动决策**
   - 所有建议都有具体数据支持
   - 量化预期节省金额和百分比
   - 行业基准对比

3. **智能交互**
   - 支持自然语言追问
   - 自动识别问题类型
   - 临时计算和假设分析

4. **知识积累**
   - 内置行业知识库
   - 支持知识更新和扩展
   - 基于成本结构智能推荐相关知识
